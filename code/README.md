# Code

This folder contains the R optimisation code for the Hospital Capacity Planning and Optimisation project.

The analysis covers:

- MILP model formulation
- Procedure room configuration decisions
- Diagnostic and therapeutic demand constraints
- Clinician availability constraints
- Setup-cost modelling
- Weekly cost breakdown
- Solver validation
- Feasibility checks
